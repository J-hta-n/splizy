## Receipts API Mock Mode

For local testing, you can mock `GET /api/receipts` and `PATCH /api/receipts`.

1. Set this in your root `.env.local`:

```bash
MOCK_RECEIPTS_API=true
```

2. Restart `npm run dev`.

When enabled:

- `GET /api/receipts?group_id=...` returns mock receipt data.
- `PATCH /api/receipts?group_id=...` validates payload, updates in-memory mock state, and returns `200`.

Disable by setting `MOCK_RECEIPTS_API=false` (or removing it).
